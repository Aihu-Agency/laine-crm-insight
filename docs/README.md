## Quick Start Guide

### For Sales Agents

1. **Login**: Navigate to the app and sign in with your email and password
2. **Dashboard**: View your pending tasks and quickly add new customers
3. **Customers**: Browse and filter your assigned customers
4. **Customer Detail**: Click any customer to view full profile, preferences, and linked properties
5. **Tasks**: Check the Todo page for pending follow-up actions

### For Managers (Admin)

In addition to sales agent features:

1. **View All Customers**: Toggle "Show everyone's Tasks" to see all team activity
2. **Import Customers**: Use the Import page to bulk upload customers via CSV
3. **User Management**: Access Settings to manage team members

## Key Features

### Customer Management
- Add new customers with contact info and property preferences
- Filter by salesperson, areas of interest, purchase timeline
- Track customer journey through the sales funnel

### Property Matching
- Properties are automatically scraped from Idealista and Laine Homes website
- Properties are linked to interested customers
- View linked properties on each customer's detail page

### Task Management
- Create follow-up actions for customers
- Track completion status
- View all pending tasks in one place

### Email Marketing
- New buyers are automatically synced to Mailchimp
- Email campaigns managed through Mailchimp dashboard

## Common Workflows

### Adding a New Customer

1. Go to Dashboard
2. Fill in First Name, Last Name, and Area of Interest
3. Click "Continue"
4. Complete the full customer form with preferences
5. Click "Save" - customer is created and (if Buyer) synced to Mailchimp

### Completing a Task

1. Navigate to Todo page or customer detail
2. Find the pending task
3. Click the checkbox to mark as complete

### Importing Customers

1. Go to Import page (Admin only)
2. Upload a CSV file with customer data
3. Map columns to customer fields
4. Review and confirm import
